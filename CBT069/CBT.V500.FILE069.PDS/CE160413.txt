*
* CODING EXAMPLE 16.4.13
*
*
This coding example illustrates the use of the DSPSERV macro
instruction to increase the size of the Data Space created in
CODING EXAMPLE 16.4.3 by 250,000 bytes.


[name]   DSPSERV EXTENT,STOKEN=DS01STOK,BLOCKS=EXPBLKS
         ...
DS01STOK DS    D           PROVIDED, RECEIVED FROM DSPSERV-CREATE MACRO
EXPBYTES EQU   250000      REQUIRED EXPANSION SIZE IN BYTES
EXPBLKS  DC    A((EXPBYTES-4095)/4096)      REQUIRED BLOCKS
