***********************************************************************
*                                                                     *
*        PDSDOC *RMF 74 SUMMARY RECORD FORMAT FROM 'DAVE74'...........* SE72REC
*                                                                     *
* NOTE THAT THE FIRST PART OF THE RECORD MAPS TO THE STANDARD SMF     *
* HEADER FORMAT SO THAT SMF ANALYSIS PROGRAMS MAY BE USED             *
*                                                                     *
* 12NOV98 DHC   CREATED BASED ON DC73REC.                             *
*                                                                     *
***********************************************************************
         SPACE 1
*                             CC
DC74REC  DS    0F                  OUTPUT SUMMARY RECORD AREA
         DC    AL2(DC74RECL)  01   PSEUDO RECORD LENGTH
         DC    AL2(0)         03   RESERVED - SET TO ZEROES
DC74FLG  DC    X'06'          05   PSEUDO SYSTEM INDICATOR
DC74TYP  DC    AL1(174)       06   RECORD TYPE
DC74TIM  DC    AL4(0)         07   TIME FROM SMF73 RECORD
DC74DAY  DC    PL4'0'         11   DATE FROM SMF73 00YYDDDF
DC74SID  DC    CL4'DCUK'      15   SYSTEM ID FROM SMF73 RECORD
         DC    AL2(1)         19   BINARY ONE FOR COUNTING
DC74PTN  DC    BL1'00'        21   PR/SM PARTITION NUMBER
DC74IST  DC    XL4'00'        22   TIME INTERVAL STARTED 0HHMMSSF
DC74DAT  DC    PL4'00'        26   DATE INTERVAL STARTED 00YYDDDF
DC74INT  DC    PL4'00'        30   DURATION OF INTERVAL  MMSSTTTF
DC74STR  DC    BL4'00'        34   TIME INTERVAL STARTED HHMMSSTH
DC74BINT DC    BL4'00'        38   INTERVAL IN BINARY HUNDREDTHS OF SEC
DC74SAM  DC    BL4'00'        42   NUMBER OF SAMPLES, WEIGHTED BY SRM
DC74DCT  DC    CL4' '         46   DEVICE NUMBER
DC74NUM  DC    BL2'00'        50   DEVICE NUMBER (4 DIGIT)
DC74LCU  DC    BL2'00'        52   LOGICAL CONTROL UNIT
DC74SER  DC    CL6' '         54   VOLUME SERIAL
DC74SSC  DC    BL4'00'        60   START SUBCHANNEL COUNT
DC74MEC  DC    BL4'00'        64   MEASUREMENT EVENT COUNT
DC74CNN  DC    BL4'00'        68   CONNECT TIME          128 MICROSEC
DC74PEN  DC    BL4'00'        72   PENDING TIME          128 MICROSEC
DC74ATV  DC    BL4'00'        76   ACTIVE  TIME          128 MICROSEC
DC74DIS  DC    BL4'00'        80   DISCONNECT TIME       128 MICROSEC
DC74QUE  DC    BL4'00'        84   NO. REQUESTS QUEUED IN IOS
DC74ALC  DC    BL4'00'        88   NO. REQUESTS WHEN DEVICE ALLOCATED
DC74ICT  DC    BL4'00'        92   NO. INCORRECT SAMPLES
DC74DVB  DC    BL4'00'        96   DEVICE BUSY DELAY TIME
DC74CUB  DC    BL4'00'       100   CONTROL UNIT BUSY DELAY TIME
DC74NDA  DC    BL4'00'       104   TOTAL NO. ALLOCATIONS FOR DEVICE
DC74MTC  DC    BL2'00'       108   NO. TAPE MOUNTS IN INTERVAL
         SPACE 1
DC74RECL EQU   *-DC74REC   = 109   OUTPUT SUMMARY RECORD LENGTH
