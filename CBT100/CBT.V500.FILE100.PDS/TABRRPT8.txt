%TUTORIAL----------  VOLUME STATUS REPORT UTILITY  ---------------------TUTORIAL
%OPTION ===>_ZCMD                                                              +
+
+ THE VOLUME STATUS REPORT PROVIDES THE USER WITH CURRENT BACKUP TAPE
+ INFORMATION FOR DASD VOLUMES.  THE FOLLINWING INFORMATION IS PRINTED
+ FOR EACH VOLUME:  LAST BACKUP DATE, MAXIMUM AND CURRENT GENERATION AND
+ CYCLE, TAPE RETPD, LAST BACKUP TYPE, EXPDT OF CURRENT GENERATION, AND
+ INFORMATIVE MESSAGES.
+
% VOLUME(S)    + SPECIFIES THAT ONLY THE DISK VOLUME(S) SPECIFIED WILL
                 BE SELECTED FOR PRINTING.  THE DEFAULT (BLANK) IS
                 DEFERRED TO THE VOLUME GROUP OPERAND.  VOLUME(S)
                 IS(ARE) SPECIFIED AS A STRING OF VOLSERS SEPARATED BY
                 COMMAS.  EXAMPLE: DLXTSO,DLXTS2,DLX170,...
+
% VOLUME GROUP(S)+ SPECIFIES THAT ONLY THE DISK VOLUME GROUP(S)
                   SPECIFIED WILL BE SELECTED FOR PRINTING.  THE DEFAULT
                   (BLANK) SELECTS ALL VOLUMES ONLINE WILL BE SELECTED
                   FOR PRINTING.  VOLUME GROUP(S) IS(ARE) SPECIFIED AS A
                   STRING OF PARTIAL VOLSER(S) SEPARATED BY COMMAS.
                   EXAMPLE: DLXTS,DLX17... WOULD SELECT:
                            DLXTSO, DLXTS2, DLX170, DLX171, DLX172, ETC.
)END
