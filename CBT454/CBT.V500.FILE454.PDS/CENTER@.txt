  Function :   This Edit macro may be used to center text within data
  ----------   line(s).

  Command syntax : CENTER
  ---------------- CENTER HELP or H

  Use :        Type CENTER on the COMMAND line and mark the range of
  -----        lines text with the "C" range command on the line
               numbers, as i.e. :
               COMMAND ===> CENTER
          ---> CC 500   --------------
               000600   |  TITLE ... |
          ---> CC 700   --------------
               000800   Subtitle : ...
               Result : the text of lines numbered 500, 600 and 700
                        will be centered. Identical process is obtained
                        with :
          ---> C3 500   --------------
