  Function :   This Edit macro may be used to plug data into a range of
  ----------   lines starting at a given column.

  Command syntax : On the COMMAND line type in
  ----------------      PLUG STRING START-COL
                   and then mark the lines with the On or OO range
                   command, or : PLUG HELP or H to view this tutorial.
                   If START-COL is not specified, the cursor position
                   will be used.

  Use :        COMMAND ===> PLUG XDCCMD    (cursor pos.) ---|
  -----        Before ************** TOP OF DATA ********** | *********
               OO 001 ... data line ...                    (UXM204A)
               000002 ... data line ...                    (UXM204B)
               OO 003 ... data line ...                    (UXM204C)
               After *************** TOP OF DATA **********************
               OO 001 ... data line ...                    (XDCCMDA)
               000002 ... data line ...                    (XDCCMDB)
               OO 003 ... data line ...                    (XDCCMDC)
                                         (at end, cursor pos.) ---|
