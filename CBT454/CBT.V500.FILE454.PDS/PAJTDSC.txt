%Tutorial - CDSCB - Change DSCB (Format-1) -------------------------------------
+  This function can be used to modify a data-set's Format-1 DSCB in a VTOC.
+
+  Follows a description of the selectable options :
%      DISPLAY+- List the Format-1 DSCB (after eventual modify) at the terminal.
+         Default is NO. Option is ignored if the data-set is not cataloged.
%      KEEP OP.+- Keep previous operands displayed on panel after an operation.
+         Default is NO.
+
+  Possible%operands+syntax :
+         SHR   CREATE(YYDDD)   EXPDT(YYDDD)   REFDT(YYDDD)
+         DSORG(XX)   RECFM(XX)   BLKSIZE(XX)   LRECL(XX)
+         ALLOC(TR/CYL/BL)   SPACE(Secondary-Amount)
+         PWR/PWW/NOP/RACF/NORACF
+         ZAP(Offset Verdata Repdata)
+         LIST/NOLIST/FULL
+  You may enter operands wraping the next two line(s).
+
+  Depress%PF3+or%PF15+to return to the%CDSCB+panel.
)END
