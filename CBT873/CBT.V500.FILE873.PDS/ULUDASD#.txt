)F FUNCTION -

  The TSO command ULUDASD will display online DASD volumes,
  and will show some of the properties of each.  A significant
  display in ULUDASD is the number of cylinders that the
  volume contains.  The ULUDASD command has also been fixed to
  show the number of cylinders in Extended Access Volumes (EAV).
  The UCBDASD program is NOT APF-authorized.

  Note:  ULUDASD has the same functionality as UCBDASD, but the
         coding uses the ULUT lookup macros from CBT File 873.
