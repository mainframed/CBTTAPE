%----------------------------- GENERAL INFORMATION -------------------- TUTORIAL
+ISL Invocation
 --------------
 To invoke ISL, select option 5.12 from the 'DELUXE PRIMARY OPTIONS MENU'.
 The first panel to be displayed is the primary menu (see below).

 ------------ System Library Subscription Service ------------------------------
 OPTION   ===>

     1 - BROWSE    - Browse or inquire library               USERID - T99999
     2 - PRINT     - Library print functions                 TIME   - 16:44
     3 - SCAN      - Scan library with keyword(s)
     4 - ADMIN     - Administrative functions (You must be authorized)
     X - EXIT      - Terminate ISL


 Enter END key to terminate ISL
