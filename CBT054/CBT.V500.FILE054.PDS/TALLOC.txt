%TUTORIAL ---------------- DATA SET ALLOCATION SUMMARY ----------------TUTORIAL
%COMMAND ===>_ZCMD

+The DATA SET ALLOCATION SUMMARY panel displays all data sets allocated
 to a user while logged on to TSO. Information displayed for each data
 set includes DDNAME, DATA SET NAME, DISPOSITION, DATA SET ORGANIZATION,
 RECORD FORMAT, LOGOCAL RECORD LENGTH, BLOCK SIZE and VOLUME SERIAL.

 Data sets with a disposition of TEMP are temporary files which are
 deleted at the end of the TSO session. Concatenated data sets are
 grouped together under a single DDNAME. Data set names which are
 greater than thirty characters in length are truncated and continued
 on the next line.

 The information displayed may be scrolled either up or down. Press
 END to terminate the display.
)END
