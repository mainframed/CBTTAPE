%--------------------- Edit DELUXE Phone List Tutorial -------------------------
%COMMAND ===>_ZCMD
+
   EXTENSION:    Key in Deluxe extension number or part of number
                 with a%*+suffix (E.G., 7033, 7*, 70*).
   PLANT:        Key in plant designation or part with a%*+suffix.
                 Acceptable values are:
                 ADM - Administration.        AEB  - Administration East.
                 DMT - Delmart Division.      FMS  - Forms Division.
                 FOX - Red Fox.               PAS  - Pascal.
                 PSB - Production Services.   REG8 - Region 8.
                 SHV - Shoreview Plant.       STP  - Saint Paul.
   LAST NAME:    Key in person's last name or part with a%*+suffix. To list
                 all names, key in%all+(E.G., JOHNSON, JOH*).
   FIRST NAME:   Key in person's first name or part with a%*+suffix.
   ROOM:         Key in a room or part with a%*+suffix. To list all rooms,
                 key in%all+(E.G., CAFETERIA, CAF*).

       Any combination of the above may be entered to produce a cross
   reference. The only exception is that NAME and ROOM entries may not
   be specified at the same time.

   Press%END+to return.
