NAccessing the DAE data set
N--------------------------
NA DAE data set that is used by one system or is shared by systems in a
Nsysplex is accessed by:
N
N-> Invoking the IPCS DAE Display panel
N   See IPCS option 3.5.
N
N-> Generating a suppressed dump
N   Use the IPCS TAKEDUMP option.
N
N-> Editing the DAE data set
N   Edit the DAE data set, using Interactive System Productivity
N   Facility (ISPF) edit.
