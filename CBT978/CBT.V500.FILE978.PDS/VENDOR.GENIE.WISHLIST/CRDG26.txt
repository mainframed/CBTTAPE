N----------------------------------------------------------
NChapter 26. PROGRAM resources
N-----------------------------
NPROGRAM resources control information for a program that is stored in
Nthe program library and used to process a transaction, or part of a
Ntransaction.
N
NYou can create PROGRAM definitions by using CEDA or DFHCSDUP, or by
Nsetting the appropriate system initialization parameters and allowing
Nprograms to be autoinstalled. See Autoinstalling programs, map sets,
Nand partition sets in Configuring for information about autoinstall
Nfor programs.
