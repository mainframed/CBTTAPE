NNull clauses
N------------
NA clause consisting only of blanks or comments or both is a null
Nclause. It is completely ignored (except that if it includes a comment
Nit is traced, if appropriate).
N
NExample:
N--------
D
D /* The above line is blank, this is only a comment. */
