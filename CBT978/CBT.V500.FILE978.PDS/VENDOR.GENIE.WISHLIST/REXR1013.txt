NChapter 10. TSO/E REXX commands
N-------------------------------
NRT
N--
NRT (Resume Typing) is an immediate command you can use to resume
Nterminal output that was previously suppressed. The RT immediate
Ncommand is available only if an exec is running in TSO/E and you press
Nthe attention interrupt key to enter attention mode. You can enter RT
Nin response to the REXX attention prompting message, IRX0920I. Terminal
Noutput that the exec generated after you issued the HT command and
Nbefore you issued the RT command is lost.
N
NSyntax
D  RT
