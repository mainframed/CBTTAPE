NAccess Method Services for Catalogs
N-----------------------------------
NChapter 9. CREATE LIBRARYENTRY
N------------------------------
NThe CREATE LIBRARYENTRY command creates a tape library entry. Use it
Nonly to recover from tape volume catalog errors.
N
NBecause access method services cannot change the library manager
Ninventory in an automated tape library, ISMF should be used for normal
Ntape library create functions.
