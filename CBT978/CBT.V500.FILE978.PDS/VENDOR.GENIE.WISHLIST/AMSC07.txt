NAccess Method Services for Catalogs
N-----------------------------------
NChapter 7. ALTER VOLUMEENTRY
N----------------------------
NThe ALTER VOLUMEENTRY command modifies the attributes of an existing
Ntape volume entry. Use this command only to recover from tape volume
Ncatalog errors.
N
NBecause access method services cannot change the library manager
Ninventory in an automated tape library, Interactive Storage Management
NFacility (ISMF) should be used for normal tape library alter functions.
