NChapter 13. DEFINE ALTERNATEINDEX
N---------------------------------
NThe DEFINE ALTERNATEINDEX command defines an alternate index. Use it
Nto show attributes for the alternate index as a whole and for the
Ncomponents of the alternate index.
N
NALTERNATEINDEX is comprised of three components:
N  DEFINE -
N    ALTERNATEINDEX( -
N      parameters -
N    ) -
N    DATA( -
N      parameters -
N    ) -
N    INDEX( -
N      parameters -
N    ) /* - */
