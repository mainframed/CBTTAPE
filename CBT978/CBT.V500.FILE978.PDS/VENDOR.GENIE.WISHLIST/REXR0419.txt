NChapter 4. Functions

NBuilt-in functions
N------------------
NDIGITS
N-------
NDIGITS returns the current setting of NUMERIC DIGITS. See the NUMERIC
Ninstruction for more information.
N
N
NSyntax:
D  DIGITS()
N
N
NExample
N-------
I  SAY DIGITS() "-> 9"
I  NUM_DIGITS = DIGITS()
I  IF (num_digits < 12) THEN DO
I    SAY "Raising NUMERIC DIGITS to 12."
I    NUMERIC DIGITS 12
I  END
I  SAY DIGITS() "-> 12"
* End of WISH.
