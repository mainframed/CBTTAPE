NTools and Service Aids
N======================
NThis information covers the tools and service aids that IBM provides
Nfor use in diagnosing MVS problems. This edition supports
Nz/OS (5694-A01).
N
NThe first chapter, Chapter 1, "Selecting tools and service aids",
Ncontains a guide on how to select the appropriate tool or service aid
Nfor your purposes. It also provides an overview of all the tools and
Nservice aids available.
N
NEach subsequent chapter covers one of the tools or service aids. While
Nchapters vary, the following topics are generally covered for each tool
Nor service aid:
N
N-> Customizing and planning information
N
N-> Starting and stopping the tool or service aid
N
N-> Receiving, formatting, and reading the output from the tool or
N   service aid.
