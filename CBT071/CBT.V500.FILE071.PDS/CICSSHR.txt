    DEAR CICS PROJECT MEMBER,

    ENCLOSED IS THE CICS SOFTWARE EXCHANGE TAPE.  THE TAPE HAS A STANDARD
    LABEL WITH VOL=SER=CICSHR AND CONTAINS FOUR DATASETS. THE FIRST
    DATASET (SHARE.CICS.INFO) CONTAINS A COPY OF THIS LETTER IN CARD IMAGE
    FORMAT (RECFM=F,LRECL=80,BLKSIZE=80). THE SECOND DATASET (SHARE.CICS.MODS)
    IS AN UNLOADED PDS CREATED BY IEBCOPY CONTAINING MOST OF THE SOFTWARE ITEMS.
    THIS PDS HAS 80 BYTE RECORDS. IPCS CLISTS FOR 1.5 AND 1.6.1 ARE IN THIS PDS.
    THE THIRD DATASET (SHARE.CICS17.CLIST) CONTAINS IPCS CLISTS FOR CICS 1.7.
    THIS PDS IS RECFM=VB,LRECL=255 WHICH IS WHY IT IS A SEPARATE FILE. LIKEWISE,
    THERE IS A FOURTH DATASET (SHARE.CICS21.CLIST) WITH IPCS CLISTS FOR VER 2.1.
    YOUR TAPE WAS CREATED ON 03/20/92. THE LAST TIME THE CONTENTS OF THE TAPE
    WERE UPDATED WAS 01/07/91.

    THE MAIN PDS HAS A MEMBER (CONTENTS) WHICH HAS A BRIEF DESCRIPTION OF EACH
    OF ITS OTHER MEMBERS. SOME OF THE MEMBERS CONSIST OF MORE THAN ONE PROGRAM,
    MAP, ETC. NO REFERENCE IS MADE TO THE CONTENTS OF FILE 3 OR 4 OF THE TAPE.
    THERE IS NO CONTENTS MEMBER FOR FILE 3 (IPCS CLISTS FOR CICS 1.7) OR FOR
    FILE 4 (IPCS CLISTS FOR 2.1).

    IF YOU HAVE ANY COMMENTS OR FEEDBACK CONCERNING THE ITEMS CONTAINED ON
