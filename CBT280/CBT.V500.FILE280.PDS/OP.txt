.cm SCRIPT REFERENCE OP
.SUBT Purpose
The ODD PAGE EJECT control word causes output to continue on an
odd-numbered page.
.SUBT Format
|
|  .OP
|
.SUBT Usage
The .OP control word causes one or two page ejects so that output
continues on an odd-numbered page.
.SUBT Defaults
This command will create a break when encountered.
.SUBT Notes
.of 4
(1) If empty-page suppression is on (.EM NO), the page number will be
incremented to the next odd page and output
will continue on the next physical page.
.sp 1
.of 4
(2) The ODD PAGE EJECT control has the same effect as the
ODD PAGE (.PD) control.
.cm End of OP
