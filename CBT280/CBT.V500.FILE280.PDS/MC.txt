.cm SCRIPT Reference MC
.SUBT Purpose
The MULTIPLE COLUMN control word restores multi-column output
after it has been suppressed by .SC (SINGLE COLUMN).
.SUBT Format
|
|  .MC
|
.SUBT Usage
If SINGLE COLUMN had been previously specified, this control word
will restore the former multiple column environment.
If SINGLE COLUMN had not been previously specified,
then a break will be caused and no other action.
.SUBT Defaults
This control word causes a break.
It takes no operands.
.SUBT Notes
.of 4
(1) This control word may not be specified within a Keep or Footnote.
.cm End of MC
