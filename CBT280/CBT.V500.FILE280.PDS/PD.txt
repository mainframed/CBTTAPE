.cm SCRIPT REFERENCE PD
.SUBT Purpose
The ODD PAGE control word causes output to continue on an
odd numbered page.
.SUBT Format
|
|  .PD
|
.SUBT Usage
The .PD control word causes one or two page ejects so that output
continues on an odd numbered page.
.SUBT Defaults
This command will create a break when encountered.
.SUBT Notes
.of 4
(1) If empty page suppression is on (.EM NO), the page number will be
incremented to the next odd page and output
will continue on the next physical page.
.cm End of PD
