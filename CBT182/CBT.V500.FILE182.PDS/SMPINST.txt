 DOCUMENTATION.  THIS IS AN AN SMP INSTALL OF PDS VERSION 8.6.

 DIRECTIONS.
    THE INSTALL DATASET FROM THE DISTRIBUTION TAPE MUST BE LOADED
    DOWN TO A DISK DATASET TO BE RENAMED SMPPREFX.PDPSORC .

    THE MEMBER SMPINST IS THEN RECEIVED.

    LIBRARIES TO BE ALLOCATED:
      80 BYTE FB  :     PDPSRC, PDPMAC, PDPPNL, PDPMSG, PDPSAMP
                        PDPXMIT
      LOAD LIBRARY:     PDPLINK, APDPLINK
      THIS LIBRARY:     PDPSORC
      TWO DLIBS   :     APDPSRC, APDPLINK

    THESE MUST BE MADE KNOWN TO THE SMP PROCEDURE, OR AS DDDEFS.
    ALSO, PDPMAC MUST BE INCLUDED IN THE SYSLIB CONCATENATION FOR
    ASSEMBLY MACRO LIBRARIES, IN ORDER FOR THE PDS ASSEMBLY TO BE
    CLEAN.

    APPROXIMATE SPACE REQUIREMENTS ON 3390 DRIVES (PLEASE ALLOCATE
    SECONDARY SPACE):  THIS IS ALSO WITH COMPRESS(ALL) TURNED ON.

      PDPSRC    -     30  TRACKS    -     5   DIRECTORY BLOCKS
      PDPMAC    -    170  TRACKS    -    35   DIRECTORY BLOCKS
      PDPPNL    -     80  TRACKS    -    80   DIRECTORY BLOCKS
      PDPMSG    -      5  TRACKS    -     5   DIRECTORY BLOCKS
      PDPSAMP   -     85  TRACKS    -    60   DIRECTORY BLOCKS
      PDPXMIT   -      5  TRACKS    -     1   DIRECTORY BLOCKS
      PDPLINK   -     10  TRACKS    -     5   DIRECTORY BLOCKS
      PDPSORC   -    450  TRACKS    -   150   DIRECTORY BLOCKS
      APDPSRC   -    450  TRACKS    -   150   DIRECTORY BLOCKS
      APDPLINK  -     10  TRACKS    -     5   DIRECTORY BLOCKS

    A USERMOD MUST BE INSTALLED TO CUSTOMIZE THE PDS OPTIONS.
    A SAMPLE USERMOD IS INCLUDED IN THIS MEMBER, SMPINST, AT
    THE BOTTOM.  THE INSTALLER SHOULD EXAMINE MEMBERS #PDSGEN
    AND #PDSGENB TO LEARN HOW TO SET THE INITIAL DEFAULTS OF PDS.

 ** NOTE:  USERMOD NOT REQUIRED IF THE "COPY #PDSGENU" STATEMENT
    IN #PDSGEN IS UNCOMMENTED (THE CURRENT DEFAULT).

    AFTER PDS 8.6 IS INSTALLED, ITS SUBCOMMAND "CONTROL GLOBALS"
    WILL DISPLAY ANY GENNED OPTION THAT DIFFERS FROM THE DEFAULTS
    SUPPLIED BY THE INSTALL PDS.  THIS MAY BE USED TO REINSTALL
    PDS SO THE OPTIONS ARE SET AS THEY WERE PREVIOUSLY.

    IN ORDER FOR THE PDS COMMAND TO HAVE ITS FULL POWER, UTILITIES
    FROM THE CBT TAPE FILES 296, 112, ET. AL. SHOULD BE INSTALLED.

    THESE NECESSARY UTILITIES MAY BE INSTALLED AS A SUBORDINATE
    FUNCTION TO THIS FUNCTION, OR THEY MAY BE INSTALLED WITHOUT SMP.
    BUT IT IS HIGHLY DESIRABLE TO HAVE THEM AVAILABLE IN THE PDS
    ENVIRONMENT.

    UTILITY COMMANDS AND PROGRAMS MUST BE AVAILABLE VIA A TASKLIB.
    USE THE THE IBM TSOLIB COMMAND, CBT STEPLIB COMMAND, OR ADD
    LIBRARY TO STEPLIB OR LINKLIST DATASET CONCATENATION.

 ** NOTE:  TSO RECEIVE MEMBER UTILXMIT TO INSTALL THESE UTILITIES.

    THE MEMBER OF THE PDPSORC LIBRARY CALLED "PDSHELP" IS THE TSO
    HELP MEMBER, WHICH SHOULD BE NAMED (OR ALIASED) PDS86, AND
    COPIED INTO A MEMBER OF YOUR SYSHELP DD CONCATENATION TO TSO.
    A BATCH UTILITY CALLED "HELPLIST" IS PROVIDED, WHICH CAN BE
    USED TO FORMAT THE TSO HELP MEMBER TO LOOK LIKE A USER MANUAL
    THAT CAN BE PRINTED.

    NEED MORE HELP?  CONTACT SAM GOLOB AT SBGOLOB@CBTTAPE.ORG
