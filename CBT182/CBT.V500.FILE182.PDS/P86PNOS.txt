)ATTR DEFAULT(%+_)
    % TYPE(TEXT) INTENS(HIGH) COLOR(WHITE) SKIP(ON)
    ¢ TYPE(TEXT) INTENS(HIGH) COLOR(YELLOW)
    + TYPE(TEXT) INTENS(LOW)  COLOR(TURQ) SKIP(ON)
    _ TYPE(INPUT) INTENS(HIGH) CAPS(ON) JUST(LEFT) COLOR(RED)
    ¬ TYPE(INPUT) INTENS(HIGH) CAPS(ON) JUST(LEFT) COLOR(RED) PAD(_)
    ¦ TYPE(INPUT) INTENS(HIGH) CAPS(OFF)
)BODY EXPAND(\\)
%TUTORIAL  +-\-\-¢PDS CAX Sort Command+-\-\-
%OPTION  ===>_ZCMD                                                             +
+
   The SORT command may be used to sort the CAX table.

   Syntax for the SORT command is as follows:
    %SORT      NUMBER/DSNAME VOLUME  ASCEND/DESCEND
+
+ +where -
      %NUMBER         -+sort by order number of CAXWA chain (default)
      %DSNAME         -+sort by catalog dataset name
      %VOLUME         -+sort by volume name
      %ASCEND/DESCEND -+sort direction (ASCEND is default)


   Notes: VOLUME and DSNAME can be used together, VOLUME will be the
          the first sort field,  DSNAME will be the second sort field.
          ASCEND/DESCEND can be used once with any other keywords.
