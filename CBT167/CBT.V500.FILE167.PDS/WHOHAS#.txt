   THIS IS A TSO (PFK6) COMMAND 'WHOHAS' THAT YOU CAN USE TO FIND
   OUT WHO HAS A DATA SET OR DISK VOLUME ENQUED.



   1).  YOU MUST ASSEMBLE THE ACCOMPANIED MEMBERS INTO A SYSTEM
        CANCATENATED LOAD LIBRARY THAT IS EXECUTABLE THROUGH YOUR
        PF6 TSO SCREEN.  BE SURE TO UPDATE THE DISK VOLUME TABLE
        'VOLTAB' IN PROGRAM WHOHAS2 BEFORE YOU ASSEMBLE ANY PROGRAM.
        USE TSO PFK6 TO CALL 'WHOHAS' ANSWER THE PROMPT AS TO WHAT
        KIND OF LIBRARY OR VOLUME YOU ARE INQUIRYING ABOUT THEN THE
        NAME OF THE VOLUME OR DATA SET YOU'RE INQUIRYING ABOUT, IT
        WILL IMMEDIATELY GIVE YOU THE ENQUED ANSWER.
