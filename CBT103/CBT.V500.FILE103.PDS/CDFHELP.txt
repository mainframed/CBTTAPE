%TUTORIAL---------------  Console Display Facility (CDF)  ------------T TUTORIAL
+
+CDF is an ISPF dialog which displays the contents of MVS graphics consoles.
 The initial console which is displayed is the master console.  Any active
 console may then displayed by entering%C=nn+to display console number "nn", or
%U=aaa+to display the console at UCB address "aaa".

 The table below lists the currently generated MVS graphics consoles.

                Console     UCB
        System   Number   Address   Location
        ------   ------   -------   --------
         XXX1      01       nnn     location of console 01
         XXX2      02       mmm     location of console 02
)END
