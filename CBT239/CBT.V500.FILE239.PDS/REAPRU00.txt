%                      'USING'  primary command
+                      ------------------------

  Primary command USING allows you define base register and specify its
position. This action is pretty similar to the assembler USING command.

  The format of primary command USING is:

 %U{SING}#Offset1/*/Offset2(Reg2)/@Name+#Reg1+#{Offset3/<cursor>}+

  You may type up to three operands. Two first must be specified, third one is
optional.

  First and second operands of command define offset or DSECT name and number
of the base register. Third operand defines the displacement, where the USING
assembler command must appear in project listing. If third operand is omitted
then Offset3 is set equal to Offset1.
