%EDITMAC  ------------------ JC2       MACRO  ------------------------  TUTORIAL
%COMMAND ===>_ZCMD                                                             +
+
+THE%JC2+MACRO IS USED TO CREATE A JOBCARD WITH DATASET INFO IN IT
+ON THE 0.2 OPTION JOBCARD TYPE IN (AMPERSAND)THIS.  "THIS" WILL BE
+REPLACED BY THE DATASET NAME.
+TYPE%JC2+ON THE COMMAND LINE AND INDICATE WHERE TO PLACE THE JOB CARD
+WITH AN "A" OR "B".
+
%COMMAND ===> JC2
+A**************************** TOP OF DATA *************************************
+000001  DATA LINE 1
+
+RESULT:
+
+***************************** TOP OF DATA *************************************
%//H562813N JOB  (AD01,E009),'DAVIS-E009-3357EB',
%//   MSGCLASS=X,REGION=2048K,
%//   CLASS=A,
%//   NOTIFY=H562813       H562813.PVTLIB.SHIPPNLS(JC2)
+000001  DATA LINE 1
)END
