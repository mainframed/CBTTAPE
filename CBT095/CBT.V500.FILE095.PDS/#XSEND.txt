EDITMAC ---------------------- XSEND     MACRO -------------------------TUTORIAL
%COMMAND ===>_ZCMD                                                             +
+
+     THE XSEND MACRO WILL INVOKE TRANSMIT COMMAND ON THE DATASET YOU ARE
+     EDITING.
+
+
%USE:
+
+           ENTER%XSEND+  ON  THE  EDIT  COMMAND  LINE  WHEN  YOU HAVE
+           COMPLETED EDITING THE DATASET.  THE TARGET DATASET WILL BE
+           'USERID.XSEND' IF MORE THAN ONE  DATASET  IS TO BE SENT, A
+           SUFFIX CHARACTER MAY BE SUPPLIED WHICH WILL BE APPENDED TO
+           THE DATASET NAME.
+
+
%EXAMPLE:+
+
+ COMMAND ===> XSEND NODEID.USER {X} OPTIONAL SUFFIX CHARACTER
+
)PROC
&KEY = .PFKEY
)END
