%EDITMAC  -------------------- CLISTBOX MACRO ------------------------- TUTORIAL
%COMMAND ===>_ZCMD                                                             +
+
+THE%CLISTBOX+MACRO IS USED TO%"BOX"+DESIGNATED CLIST LINES MAKING  THEM
+COMMENTS. IT CAN BE USED FOR COMMENTS OR TO NOP CODE LINES.
+USE:
+ON THE COMMAND LINE TYPE IN%CLISTBOX+AND USE THE CC OR CN RANGE COMMANDS
+TO INDICATE THE LINES THAT YOU WANT%"BOXED".
+
%COMMAND ===> CLISTBOX
+***************************** TOP OF DATA *************************************
+000010     SET TYPE = COMMAND
+CC0020    CONTROL MSG LIST CONLIST SYMLIST
+CC0030    WRITE SET UP DONE
+000040    WRITE PROCESS TERMINATING
%RESULT:
+***************************** TOP OF DATA *************************************
+000010     SET TYPE = COMMAND
+000020/**********************************************************************
+000030/*  CONTROL MSG LIST CONLIST SYMLIST                                  *
+000040/*  WRITE SET UP DONE                                                 *
+000050/**********************************************************************
+000060    WRITE PROCESS TERMINATING
)PROC
&KEY = .PFKEY
)END
