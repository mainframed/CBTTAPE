%EDITMAC  -------------------- LJUST  MACRO  -------------------------  TUTORIAL
%COMMAND ===>_ZCMD                                                             +
+
+THE%LJUST+MACRO IS USED TO LEFT JUSTIFY TEXT WITHIN A DATA SET.
+
+USE:
+FIRST ON THE LINE NUMBERS OF THE DATA SET INDICATE A RANGE OF THE LINES
+TO BE "JUSTIFIED" , "CC" TO "CC" OR "CNN" (NN BEING NUMBERS).  THEN ON
+THE COMMAND LINE TYPE IN LJUST. (SCROLLING IS NOT ALLOWED AFTER TYPING
+IN LJUST UNTIL THE LJUST COMMAND IS COMPLETE).
+
%COMMAND ===> LJUST
+***************************** TOP OF DATA *************************************
%=BNDS> <                                                            >
+000001          DATA LINE 1
+CC0002               DATA LINE 2
+CC0003                  DATA LINE 3
+
+RESULT:
+***************************** TOP OF DATA *************************************
%=BNDS> <                                                            >
+000001          DATA LINE 1
+000002 DATA LINE 2
+000003 DATA LINE 3
)PROC
&KEY = .PFKEY
)END
