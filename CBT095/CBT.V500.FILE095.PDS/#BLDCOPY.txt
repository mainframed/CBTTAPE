%EDITMAC  -------------------- BLDCOPY MACRO -------------------------  TUTORIAL
%COMMAND ===>_ZCMD                                                             +
+
+THE%BLDCOPY+MACRO IS USED TO CREATE AN IEBCOPY SELECTION LIST.
+
+USE:
+ONTE COMMAND LINE TYPE IN%BLDCOPY+AND THE NAME OF THE PDS WHICH IS TO
+TO BE COPIED.  PLACE A "A" OR "B" ON THE LINE NUMBER WHERE YOU WANT THEE
+LIST INSERTED PRESS ENTER.
+
%COMMAND ===> BLDCOPY  MY.PDS
+***************************** TOP OF DATA *************************************
+000130  //SYSIN DD *
+A00140  COPY INDD=IN1,OUTDD=OUT1
+
+RESULT:
+***************************** TOP OF DATA *************************************
+000130  //SYSIN DD *
+000140  COPY INDD=IN1,OUTDD=OUT1
+000150     S M=CENTER
+000160     S M=CUT
+000170     S M=JC
+000180     S M=LJUST
+000190     S M=PASTE
)PROC
&KEY = .PFKEY
)END
