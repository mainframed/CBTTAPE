.cm SCRIPT REFERENCE PV
.SUBT Purpose
The EVEN PAGE control word causes output to continue on an
even numbered page.
.SUBT Format
|
|  .PV
|
.SUBT Usage
The .PV control word causes one or two page ejects so that output
continues on an even numbered page.
.SUBT Defaults
This command word will create a break when encountered.
.SUBT Notes
.of 4
(1) If empty page suppression is on (.EM NO), the page number will be
incremented to the next even page and output
will continue on the next physical page.
.cm End of PV
