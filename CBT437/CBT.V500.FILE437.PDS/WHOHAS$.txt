WHOHAS

Function

This program will retrieve all outstanding GRS ENQ's and return them in
an ISPF table. It is possible to limit the amount of displayed rows by
means of wildcards in the various columns.


Using WHOHAS

CALL *(JJ$LWA)

or create an ISPF command table entry with PGM(JJ$LWA)


Installation requirements

JJ$LWA needs to be installed in ISPLLIB or LINKLST library, if it is
installed in any other library then in must be called specifically (ie.
CALL 'IBMUSER.LOAD(JJ$LWA)')

JJ$PWA must be installed in an ISPPLIB library.
