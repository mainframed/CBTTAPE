PARMTZ

Function

This OMVS program will retrieve the UTC/local time offset from MVS. It
will return the offset in a manner compatible for use with the TZ shell
variable.


Using PARMTZ

The following example will set the TZ variable to the UTC/Local offset
from MVS.

In etc/profile or $HOME/.profile:

export TZ=CET`/usr/local/bin/parmtz`



Installation requirements

parmtz needs to be installed in a HFS.
