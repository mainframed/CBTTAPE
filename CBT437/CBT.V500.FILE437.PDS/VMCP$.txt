VMCP

Function

This subsystem will add a CP command to MVS in order to issue VM
commands thru SVC34.


Defining VMCP to the system

The subsystem needs to be defined as following:

SYS1.PARMLIB(IEFSSNxx):

VMCP,VMCPSSIN,'CP Q STOR   (or any other command)'

Using VMCP

Simply use CP as a MVS command either from an operator console, SDSF,
netview or any other means to issue standard MVS operator commands.

Installation requirements

VMCP needs to be installed in a linklist or lpalist library.

DOWNLOAD

Sourcecode is distributed in distribution file vmcp.zip
