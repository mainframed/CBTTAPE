This is the partitioned dataset for the Format/Edit routines.

$FMTDOC  - documentation of the history of this collection
$INDEX   - this text
EYEC     - ALC macro
FMT$ASM  - jobstream to assemble/link the routines
FMT$INST - jobstream to allocate load library & submit FMT$ASM
FMT$IVP  - jobstream to run the installation verification program
FMTIVP   - COBOL Installation Verification program
IFMTFILL - input field must be filled
IFMTFLPT - input field floating point format
IFMTHEX  - input field hex digits
IFMTLJST - input field left justify
IFMTMASK - input field extract using mask
IFMTNIB  - input field no imbedded blanks
OFMTHEX  - output field printable hex digits
OFMTMONY - output field monetary field as words
