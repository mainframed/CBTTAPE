%TUTORIAL ----- PANEXEC LIBRARY UTILITY - RENAME ELEMENT--------------- TUTORIAL
%OPTION  ===>_ZCMD                                                             +
+
+  The%rename element+function is used to rename a element of a PANEXEC
   library.  (Use the%element list+option if you wish to rename several
   elements.)

   To rename a single element, fill in the following fields of the library
   utility panel:
       - Enter%R+in the option field.
       - Enter the%library information in the appropriate fields.
       - Enter the old element name in the%element+field.
       - Enter the new element name in the%new elt+field and/or
       - Enter the new group name in the%new grp+field

)PROC
   &ZUP = pxp000a
)END
