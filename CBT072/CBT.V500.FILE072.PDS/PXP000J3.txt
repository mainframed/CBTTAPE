%TUTORIAL -------------- PANEXEC LIBRARY MAINTENANCE - CREATE --------- TUTORIAL
%OPTION  ===>_ZCMD                                                             +
+
     The library create option should be executed on an as needed basis
to create new PANEXEC libraries or re-create damaged PANEXEC libraries.
A PANEXEC group name must be specified on the main PANEXEC library
utility panel to establish a default group name for the PANEXEC library.

 The following fields must be specified on the library maintenance panel
   Volume Serial   - the volser the PANEXEC library is to reside on
   Space Allocation- number of cylinders to be allocated to the PANEXEC
                     library
   Maximum Elements- maximum number of elements on the PANEXEC library

 The following topics are presented only if selected by number:

        1 - CREATE JCL AND PANEXEC PROCEDURE
)PROC
   &ZSEL = TRANS(&ZCMD
                1,*pxp000jd
                )
   &ZUP = pxp000j1
)END
