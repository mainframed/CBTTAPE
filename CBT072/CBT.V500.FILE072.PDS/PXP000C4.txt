%TUTORIAL -- PANEXEC-ELEMENT SELECTION LISTS - GENERIC NAME MATCHING -- TUTORIAL
%COMMAND ===>_ZCMD                                                             +
+
   A generic element name allows you to specify a subset of the elements
   in a PANEXEC library.

   The generic name is specified as:

          %string*+

   where%"string"+is a character string of length n, where 0 < n < 8.
   For an element to match a generic name, the first n characters of the
   element name must match the n characters of%"string".+ The
   asterisk%(*),+if used, implies a match of the remaining characters in
   the element name. If an element matches the generic name, it is
   included.  Omitting the asterisk causes%string+to be used as an exact
   element name match.

)PROC
   &ZUP   = pxp000c2
)END
