%TUTORIAL ----  A B L  JOB Queue Processor: PRINT JOB Option  -------- TUTORIAL
+
+ Command ===>_ZCMD
+
+  Selecting Option "P" results in the scheduling of the JOB to be printed.
+ Typically, this option is requested to print a JOB that is in the HOLD
+ class (Q). JOBS in the held output class are candidates for viewing on
+ the terminal and remain held until explicit action is taken to either
+ print or delete the JOBS.
+
+
+ Press%ENTER+ to continue with tutorial
+
+ Press%PF3+to return
+
)INIT
 .HELP = TUJOBQ00
)PROC
  &ZPRIM = NO
  &ZHTOP = TUJOBQ01
  &ZHINDEX =  TUJOBQ00
)END
