)F FUNCTION -
  THE LISTVTOC COMMAND DISPLAYS THE DATA SET NAMES FROM THE
  VTOC OF A DASD VOLUME, UNSORTED, IN THE ORDER THAT THEY
  APPEAR IN THE VTOC.
)X SYNTAX  -
         LISTVTOC  'VOLUME'  LEVEL('LEVEL')  OUTFILE('FILENAME')   MSS
  REQUIRED - VOLUME
  DEFAULTS - ALL DATA SET NAMES ARE DISPLAYED ON THE TERMINAL.
  ALIAS    - NONE
)O OPERANDS -
  'VOLUME' - AN ON-LINE DIRECT ACCESS VOLUME SERIAL.
))MSS      - IF THE VOLUME IS NOT CURRENTLY ON-LINE, THIS
             KEYWORD CAUSES THE COMMAND TO ATTEMPT TO ALLOCATE
             THE VOLUME WITH UNIT=3330V.
))LEVEL('LEVEL') - ONLY DATA SET NAMES BEGINNING WITH THE
             CHARACTERS IN PARENTHESES WILL BE DISPLAYED.
))OUTFILE('FILENAME') - THE DATA SET NAMES WILL BE PLACED IN A
             DATA SET PRE-ALLOCATED TO THE SPECIFIED FILENAME
             INSTEAD OF BEING DISPLAYED ON THE TERMINAL.
))ISAM     - ONLY ISAM DATA SETS WILL BE DISPLAYED.
))VSAM     - ONLY VSAM DATA SETS (INCLUDING SPACES, PAGESPACES)
             WILL BE DISPLAYED.
