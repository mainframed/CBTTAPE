                            XMITIP Overview



XMITIP is an OS/390 application written in OS/390 REXX with an
optional OS/390 TSO ISPF frontend. In the package is full
documentation (in Adobe Acrobat PDF format, Microsoft Word, RTF,
ASC, HTML, and TXT formats).

XMITIP is designed to be an SMTP mailer package for use on OS/390
in a TSO, TSO/ISPF, or Batch environment.

Check out Andy Robertson's page "How we set up SMTP and XMITIP
using a non-standard code page".

Note that XMITIP is an evolving tool that evolves in small
increments based upon the comments and suggestions of its users.

Join the XMITIP Mutual Support Listserv by sending an e-mail to
XMITIP Listserv

-------------------------

Some of its capabilities are:

  1. Send messages to one or more recipients
  2. Send using CC and BCC addressing
  3. Specify both FROM and REPLY-TO addresses
  4. Send files as attachments
       1. plain text
       2. converted to Rich Text Format (RTF)
            1. supports ASA carriage control
       3. convert to HTML (very basic)
            1. includes support for background and text color
               specifications
       4. binary (no conversion of the data)
            1. use for ZIP files
       5. Convert files (1-4) and then ZIP using InfoZip, ISPZIP
          or PKZIP/MVS
       6. convert files to PDF (Adobe Portable Document Format)
          readable using Acrobat
  5. Optional ISPF Dialog
       1. address book to select addressing from
       2. list of dataset for file attachment specifications
