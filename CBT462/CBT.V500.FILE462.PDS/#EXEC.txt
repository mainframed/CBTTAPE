)ATTR DEFAULT(%+_)
)BODY Expand(//)
%Tutorial /-/  EXEC  /-/ Tutorial
%SELECTION ===>_ZCMD                                                           +
%                     ---------------------------------
                      |     Edit Macro  "EXEC"        |
                      ---------------------------------
+
    This Edit Macro will execute the member you are currently editing.  The
    real benefit of this macro is that you do not have to save the changes.
    It will execute the changes without saving them.

%SYNTAX:+   EXEC SAVE
+
+           SAVE is an optional keyword. When requesting the SAVE option,
+                this edit macro will first save all your changes and then
+                execute the CLIST.
+
)PROC
     &ZUP = #EMCH
)END
