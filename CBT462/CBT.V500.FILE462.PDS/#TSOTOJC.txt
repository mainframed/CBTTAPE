%Tutorial /-/  TSOTOJCL  /-/ Tutorial
%SELECTION ===>_ZCMD                                                           +
%                   -----------------------------------
                    |     Edit Macro  "TSOTOJCL"      |
                    -----------------------------------
+
      This Edit Macro will insert code into the PROC section of a panel.
      The purpose of the code is to translate a dataset name entered in TSO
      syntax to JCL syntax.

%SYNTAX:+   TSOTOJCL var1 var2
+
+   "var1"  is the variable name (without an ampersand) containing the dsname
+           in TSO syntax.
+
+   "var2"  is the variable name (without an ampersand) that will contain the
+           dsname in JCL syntax.
+
)PROC
     &ZUP = #EMCH
)END
