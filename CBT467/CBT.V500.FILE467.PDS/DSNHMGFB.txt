%OPTION  ===>_ZCMD                                                             +

%                   -------------------------------------
                    |   DSM "EXPAND" PRIMARY COMMAND    |
                    -------------------------------------
+
    The normal display allows a user to enter the Dataset name with an
    optional Volser for every dataset maintained in the list. There
    are three other fields that are maintainable for each dataset.

    These fields are the Edit Profile, Edit Imacro and a Comment
    field.  The Expand Command allows the user to request the type of
    display to be provided.

%   EXPAND ON+ (default) will provide an expanded display allowing the
               user to update the Profile, Imacro and Comment fields.

%   EXPAND OFF+will provide a display with only the DSNAME and VOLSER
               fields supplied.
+
)PROC
   &ZUP = DSNHMGF
)END
