                      | THE 'KBR' COMMAND (continued) |

+
     If the DDname points to a concatenation, a Dataset List will be
     displayed. The user can then select the file he wishes to Browse.
     For Example:  KBR SYSPROC

  -BROWSE BY DDNAME --------------------------------------------------------
  ...
  CMD   DATASETS ALLOCATED TO SYSPROC:
  --    -------------------------------
  --    'LIB 1'
  --    'LIB 2'
  ...
     Finally, in the above example, if a member name is specified, the list
     of datasets will be limited to those containing that member. If only
     one library contains the member, you will enter BROWSE directly on that
     member. You may code the FIRST operand and bypass the display
     completely.
     For Example:  KBR SYSPROC FIRST
)PROC
 &ZUP = $KBR
)END
