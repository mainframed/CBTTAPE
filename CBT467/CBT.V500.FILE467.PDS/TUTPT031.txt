%TUTORIAL ------  TUTORIAL PRINT - SPECIFIYNG PANEL LIBRARIES  -------- TUTORIAL
%COMMAND ===>_ZCMD                                                             +
+

      When you want to specify a panel library with a non-standard ISPF
   library name, enter it under the heading:

       OTHER PARTITIONED DATA SET:
          DATA SET NAME %===>+

      You may enter any fully qualified data set name by enclosing it in
   apostrophes.  Example:  %DATA SET NAME ===> 'sys1.maclib'+
   If you omit the apostrophes, your TSO prefix is left-appended to the
   data set name.

      Whenever an "other" data set name is entered, it is used even if
   an ISPF library is also entered.
