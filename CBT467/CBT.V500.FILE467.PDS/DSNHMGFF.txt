%TUTORIAL -------------  DATASET MANAGEMENT FACILITY  ----------------- TUTORIAL
%OPTION  ===>_ZCMD                                                             +
+
%                   -------------------------------------
                    |   DSM "CREATE" PRIMARY COMMAND    |
                    -------------------------------------
+
    The CREATE command allows a user to copy the current list of datasets
    into a special save area. The create function requires the save area
    does not already exist. The COPY function can then be used to retrieved
    the saved dsn list.

%   CREATE label

+   will save the list of datasets as list "label" where "label" is from
+   one to three bytes.
+
+   CREATE can be abreviated as CR.
+
)PROC
   &ZUP = DSNHMGF
)END
