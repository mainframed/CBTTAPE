BITMAP   - CALLABLE SUBROUTINE TO EXPAND 8 BITS TO 8 BYTES CHARACTER.
BMSS     - FAIRLY FAST TEXT SEARCH PROGRAM.
BOMB     - SUBROUTINE TO PRODUCE AN '0CB' ABEND. DIAGNOSTIC TOOL.
COB1     - COBOL/DB2 CROSS REFERENCE BASED ON USERID.
COB2     - COBOL/DB2 CROSS REFERENCE BASED ON TABLENAME.
COPYANY  - VERSATILE FILE TO FILE COPY: FB->VB, VB->FB, VB->VB, FB->FB,
         - PADS OR TRUNCATES BASED ON OUTPUT DCB. DISPLAYS DCB, COUNTS.
         - ADDED A PARM= OPTION FOR FILL CHARACTER.
CPUID    - AUTHORIZED PGM TO DISPLAY CPC ID. OBSOLETED BY: D M=(CPU)
CSTRACE  - SAMPLE OF USING SNAPSHOT MACRO TO CREATE A DEBUGGING TRACE.
         - (YES, IT LOOKS LIKE IMS CALLS BUT ITS ACTUALLY AN IMS CLONE).
CVD      - CALLABLE SUBROUTINE TO DUMP 80 BYTES IN VERTICAL HEX FORMAT.
CVX      - CONVERT TWO BYTES DISPLAY FORMAT HEX TO ONE BYTE BINARY.
DBALPHA  - PRODUCES POSTERS WITH MAX OF 6 LINES OF 13 CHARACTERS EACH.
DBWAIT   - MEANT TO INTERACT WITH AUTOMATION SOFTWATE (AO, NETVIEW) TO
         - CONTROL STEP BY STEP EXECUTION OF BATCH JOBS.
DYNAM    - CBT TAPE FILE089, INCLUDED HERE FOR CONVENIENCE ONLY.
ERASER   - OPEN/CLOSE A CARD IMAGE FILE TO EMPTY IT, OPTIONALLY INSERT
         - ONE DATA LINE FROM PARM=.
FLATLIB  - CONVERTS A TEXT PDS INTO A FLAT FILE WITH THE MEMBERNAME AS
         - AS A PREFIX. EASY TO BUILD XREFS WITH SAS/EASYTRIECE/ETC.
LNKLST00 - MULTI STEP JCL TO PRODUCE A CROSS REFERENCE OF LPA/PROG00
         - MEMBERS. SORTED BY MEMBERNAME AND LOADLIB. PRIMITIVE ISRDDN.
MEMLIST  - EXPANDS A LIST OF PDS NAMES INTO MEMBERNAMES AND ATTRIBUTES.
         - SHORT FORM FOR TEXT PDS, DEFAULT FOR LOAD LIBRARIES. DYNAM
MIGRATOR - USES MGCRE AND VIRTUAL CONSOLE MACROS TO ISSUE HSM MIGRATE
         - COMMANDS FOR A LIST OF DATASETS AND LOG SYSTEM RESPONSE MSGS.
         - AMODE(31), RMODE(24), AC(1). MESSY DUE TO AR, IO, AMODE31...
NATSUB   - SUBROUTINE TO SUBMIT JCL TO DYNALLOC INTERNAL READER.
PARSIFAL - PARSER TO STRIP DSNAMES FROM FROM FORMATS LIKE (LPALST00)
QDUMP80  - FORMATTED DATASET DUMP SHOWS CHARACTER/HEX IN 80 COL LISTING.
         - DOES START-END AND LAST RECORDS. WILL TRANSLATE ASCII ALSO.
R******* - JCL TO RUN THE PROGRAMS.
REGS     - REGISTER EQUATE MACRO.
SLEEPY   - WAITS FOR NN SECONDS. 99 MAX.
TODAY    - SUBROUTINE RETURNS DAY OF WEEK FROM TIME MACRO. VALID TO Y4K.
VOLLIE   - VARY A LIST OF VOLSERS (NOT ADDRESSES) ONLINE/OFFLINE.
VSAMZAP  - SUPERZAP UTILITY FOR ESDS VSAM, ORIGINALLY USED FOR IMS,
         - ALSO WORKS ON KSDS BUT CAN'T CHANGE KEYS.
